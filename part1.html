JavaScript Fundamentals

Looking back... (Prerequisites)

By now you've dipped your toes (or fingers) in the JavaScript language by doing Codecademy's JavaScript tutorial (or an equivalent). Ideally, you have had some exposure to the syntax for the basic data structures (objects and arrays), control flow constructs (loops and conditionals), and encapsulating code into functions.

Looking forward...

In this class we are going to dive deeper and look closely at these concepts and solidify our understanding of the fundamentals so that you have a good foundation to build upon as you move onto more advanced topics in JavaScript.

We will also cover some good engineering practices like debugging and pseudocoding and we'll start to get into the habit of thinking algorithmically.

And if you join us for Part 2: JS Functions of this class, we'll explore the power of functions in JavaScript! We will cover concepts such as scope, closures, higher order functions, callbacks, and dive into the Underscore.js library.
